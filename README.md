# /launch demo

A 5-minute demo deck for `/launch` — a Claude Code skill that takes a built-but-unbranded software product from working prototype to launch-ready.

**Live:** https://humayunk.github.io/launch-demo/

## What's in here

- `index.html` — the demo deck (open in browser, use ← → to navigate)
- `moodboard.html` — brand mood board with reference brands and palette directions
- `system.html` — full design system mockups with light/dark toggle
- `landing.html` — landing page for Cortex (built with /launch)
- `icons/preview.html` — icon concept exploration
- `SKILL.md` — the actual skill file
